=============================================================================
                    SCRIPT PULIZIA SAMSUNG GALAXY BOOK 4 EDGE
=============================================================================

DESCRIZIONE:
Questo script automatizza la pulizia del Samsung Galaxy Book 4 Edge rimuovendo
tutte le app Samsung non necessarie per chi non possiede un telefono Galaxy,
oltre a disinstallare OneDrive.

FILES CREATI:
1. Cleanup-GalaxyBook.ps1       - Script PowerShell principale
2. Esegui-Pulizia-GalaxyBook.bat - File batch per esecuzione facile
3. README-Pulizia-GalaxyBook.txt - Questa documentazione

=============================================================================
                                 UTILIZZO
=============================================================================

METODO 1 - Esecuzione facile (CONSIGLIATO):
1. Fai doppio clic su "Esegui-Pulizia-GalaxyBook.bat"
2. Conferma l'esecuzione come amministratore
3. Lo script farà tutto automaticamente

METODO 2 - Esecuzione manuale:
1. Apri PowerShell come Amministratore
2. Naviga nella cartella dello script: cd C:\Users\massi
3. Esegui: .\Cleanup-GalaxyBook.ps1
